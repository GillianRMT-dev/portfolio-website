# Portfolio Website

A multi-page HTML portfolio website showcasing web development projects and skills. Built as a comprehensive demonstration of HTML5 fundamentals, semantic markup, forms, and web accessibility principles.

## Project Structure

```
portfolio-website/
├── index.html          # Homepage with welcome message and featured projects
├── pages/
│   ├── about.html      # About page with personal information
│   ├── projects.html   # Full projects showcase page
│   └── contact.html    # Contact form page
├── assets/
│   ├── robot-face.png
│   ├── prices-visualization.png
│   ├── escape-room.png
│   ├── about-image-one.png
│   └── about-image-two.png
└── README.md
```

## Pages Overview

### Home (index.html)

- Welcome section with brief introduction
- Featured projects showcase with images and descriptions
- Quick navigation to About page and Projects page
- Call-to-action buttons for exploring more content

### About (pages/about.html)

- Personal introduction and background
- Profile images
- Skills and experience highlights

### Projects (pages/projects.html)

- Complete portfolio of web development projects
- Project thumbnails with descriptions
- Featured projects include:
  - CSS Robot Face
  - Prices Visualization Calculator
  - Escape Room Project

### Contact (pages/contact.html)

- Comprehensive contact form with validation
- Personal information fields (name, email, bio)
- Location selection (country and state/province)
- Source tracking (how visitors found the site)
- Form includes HTML5 validation attributes

## Features Implemented

### HTML Structure

- ✅ Semantic HTML5 elements (`<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<figure>`, `<footer>`)
- ✅ Proper document structure with doctype and meta tags
- ✅ Consistent navigation across all pages
- ✅ Organized folder structure for pages and assets

### Content Elements

- ✅ Logical heading hierarchy (h1-h3)
- ✅ Paragraphs and text content
- ✅ Unordered lists for navigation
- ✅ Images with descriptive alt text
- ✅ Internal navigation links
- ✅ Figure elements with captions for project images

### Forms & Validation

- ✅ Multi-fieldset contact form
- ✅ Various input types (text, email, textarea, select, radio)
- ✅ HTML5 validation attributes:
  - `required` fields
  - `type="email"` for email validation
  - `minlength` and `maxlength` for textarea
  - `placeholder` text for user guidance
- ✅ Proper form controls (submit and reset buttons)
- ✅ Organized fieldsets with legends

### Accessibility

- ✅ Alt text on all images
- ✅ Form labels associated with inputs
- ✅ Semantic HTML for better screen reader support
- ✅ Proper heading hierarchy
- ✅ Descriptive link text

### Best Practices

- Valid HTML5 markup
- Clean, readable code structure
- Consistent formatting and indentation
- Proper file organization
- Responsive viewport meta tag

## How to View

1. Clone the repository:

   ```bash
   git clone https://github.com/GillianRMT-dev/portfolio-website.git
   ```

2. Navigate to the project directory:

   ```bash
   cd portfolio-website
   ```

3. Open `index.html` in your web browser

Alternatively, you can view individual pages directly by opening them in any modern web browser.
